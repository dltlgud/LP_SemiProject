### 📖 프로젝트 소개
LP 레코드를 사랑하는 수집가들을 위한 온라인 쇼핑몰 플랫폼입니다. 
단순한 구매를 넘어, 레코드판 특유의 감성을 전달할 수 있는 UI와 안정적인 데이터 관리 시스템 구축을 목표로 합니다.

### 🛠 주요 기술 스택 (Tech Stack)
- **언어:** Java, JavaScript, SQL (Oracle)
- **도구:** VS Code, Git
- **핵심 역량:** 관계형 데이터베이스 설계(ERD), RESTful API 구현, 객체지향 설계

### ✨ 주요 기능 (Key Features)
- **[상품 관리]** 아티스트, 장르, 발매일 기반의 상세 필터링 및 검색
- **[주문 시스템]** 장바구니 담기 및 실시간 재고 연동 결제 프로세스
- **[사용자 경험]** 아날로그 테마를 적용한 직관적인 웹 인터페이스
- **[데이터 시각화]** UML 및 시퀀스 다이어그램을 통한 체계적인 로직 관리

### 🗄 데이터베이스 구조
프로젝트의 핵심인 LP 상품 정보와 주문 이력을 효율적으로 관리하기 위해 설계되었습니다.
- `USER_TABLE`: 회원 등급 및 개인정보 관리
- `PRODUCT_TABLE`: 레코드 상세 정보 및 재고 상태
- `ORDER_TABLE`: 주문 및 배송 상태 추적
