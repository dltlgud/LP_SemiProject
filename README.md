## 🚀 실행 방법

### 사전 요구사항
- Java 11 이상
- Apache Tomcat 9.x
- Oracle DB (또는 MySQL)

### 로컬 실행

```bash
# 1. 저장소 클론
git clone https://github.com/dorapizza1234/LP_SemiProject.git
cd LP_SemiProject

# 2. DB 설정 (src/main/resources/db.properties 수정)
# db.url=jdbc:oracle:thin:@localhost:1521:xe
# db.username=your_username
# db.password=your_password

# 3. Tomcat에 프로젝트 배포 후 실행
# Eclipse/IntelliJ에서 Run on Server 또는 mvn tomcat7:run
```

### 접속
```
http://localhost:8080/LP_SemiProject
```
